# Tech Quiz Application

This is a full-stack Tech Quiz platform consisting of multiple modules:

## 🔗 Project Repositories

- Frontend: https://github.com/iamcelina1234/quiz-frontend
- Backend: https://github.com/iamcelina1234/quiz-backend
- Admin Panel: https://github.com/iamcelina1234/quiz-admin

## 🚀 Features

- Interactive quiz system with multiple-choice questions  
- Real-time score tracking  
- Admin panel to manage questions  
- Backend APIs for data handling  

## 🛠️ Tech Stack

- Frontend: HTML, CSS, React
- Backend: Node.js, Express.js  
- Database: MongoDB  

## 📌 Description

This project is designed to help users test their technical knowledge through quizzes and track their performance efficiently.
